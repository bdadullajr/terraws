# Terraform +  AWS provisioning

Tools version needed to install. 
```
- Terraform v0.11.13
- git version 2.14.5 
```

Prerequisites
```
	1.) Installed terraform version - https://releases.hashicorp.com/terraform/
	2.) Git to clone from github
	3.) AWS Account with enough privileges
		- Access key ID
		- Secret access key
```
